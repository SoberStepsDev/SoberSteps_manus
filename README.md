# SoberSteps

Aplikacja Flutter do wspierania trzeźwości (SoberSteps).

## Dokumentacja w repozytorium

- **[docs/](docs/)** — checklista launch, spec Recovery+, handoff workflow, polityka zależności  
- **[android/KEYSTORE.md](android/KEYSTORE.md)** — podpisywanie buildów Android

## Szybki start

```bash
flutter pub get
flutter analyze --no-pub --congratulate
flutter test
```

Konfiguracja środowiska: `assets/config.env.example` (właściwy `config.env` poza Gitem).
